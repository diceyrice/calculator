# Calculator
A calculator with basic arithmetic operations using HTML, CSS, and JS.

## Features
* Support for addition, subtraction, multiplication, and division.
* Clear button to wipe the slate clean.
* Clears the calculator automatically when inputting a new number after the equals button is pressed.
* Currently only works with mouse input.
